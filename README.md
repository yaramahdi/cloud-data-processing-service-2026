# Cloud-Based Distributed Data Processing System

This project implements a cloud-based distributed data processing system using Google Cloud Platform.

## Technologies
- Google Cloud Dataproc (Apache Spark)
- Google Cloud Storage
- Google Cloud Run
- PySpark
- Flask

## Features
- Run Spark jobs on a Dataproc cluster
- Compare performance with different numbers of worker nodes (1, 2, 4, 8)
- Simple web interface to submit jobs
- Results stored in Cloud Storage

## Spark Jobs
- Descriptive statistics
- Linear Regression
- Decision Tree Regression
- K-Means Clustering
